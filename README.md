# Customer Segmentation Using K-Means Clustering

## Objective
To segment customers into distinct groups using the K-Means clustering algorithm based on their purchasing history.

## Description
This project applies K-Means Clustering to identify patterns in customer data and group customers with similar characteristics.

## Input Features
- Customer attributes (e.g., Age, Annual Income, Spending Score)

## Output
- Customer clusters

## Algorithm
- K-Means Clustering (Unsupervised Learning)
